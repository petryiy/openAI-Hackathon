# Product evals

Each case should test the same five outcomes:

1. factual/canonical explanation is correct;
2. plot causally depends on the concept;
3. choices map to distinct learner hypotheses;
4. branch visualization matches the selected teaching strategy;
5. transfer task changes surface context but preserves the relationship.

`moonbase-projectile-motion.json` is the release-blocking baseline. Future cases should include conditional probability and a geometry transformation so improvements do not overfit projectile motion.
